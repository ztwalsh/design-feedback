# F

---

| Term | Definition | Usage Guidelines |  
| **field mapping** | Connecting data from one step to use in another step. | Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase; when capitalized, use sentence case. |  
| **Filter by Zapier** | App created by Zapier that stops Zaps from running unless conditions are met. | Capitalize the app name even when using it without the “by Zapier” suffix. |  
| **filtered** | Status when a Zap doesn't run because conditions weren't met. | Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. |  
| **flood protection** | Setting that limits how many new events can be sent through your Zap before being paused. | Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase; when capitalized, use sentence case. |  
| **Formatter by Zapier** | App created by Zapier for changing data format (dates, text, etc.). | Capitalize the app name even when using it without the “by Zapier” suffix. |  
| **formulas** | Tables feature that allows you to perform math operations and compute results based on data from your fields. | Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. |  
| **Functions** | Code workflows in a web IDE. | Functions is a Zapier product, so it should be capitalized when referred to. Use lowercase when referring to it as an asset. |
