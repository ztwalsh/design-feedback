| Term | Definition | Usage Guidelines | Related terms |  
| **[App] by Zapier** | Apps or tools created by Zapier. | Use when talking about specific apps, like Formatter by Zapier. When collectively mentioning "by Zapier" apps, use the term Built&hyphen;in tools. When space is limited, you can drop the “by Zapier” suffix. | built&hyphen;in tools |  
| **action** | What your Zap does after it's triggered (like sending an email or creating a contact). | Do not capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. | action event |  
| **action event** | The specific activity that happens when your Zap runs. | Do not capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. | action |  
| **admin** | User role with additional account permissions. | Do not capitalize unless used to start a sentence or in conjunction with the user's name. For example, Admin Max Jackson. | super admin |  
| **Admin Center** | A centralized set of controls and settings to help admins manage their organization’s account. It is only available to owners and admins. | Admin Center is a proper noun and should be capitalized. |  |  
| **Agents** | AI assistants you can create with Zapier. | Agents is a Zapier product and should be capitalized when referring to it. Use lowercase when referring to it as an asset. |  |  
| **alerts** | A feature that allows users to set up notifications for important Zap updates. | Do not capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. |  |  
| **app permissions** | Settings where admins and owners control which apps and actions members of the account can access and use. | Sunset category name: allowed and restricted apps |  |  
| **alpha** | The app, feature, or product is still in development and is only available by invite. | When using in a tag or label use all caps. [Example](https://cdn.zappy.app/f113dee6e9f6a6627a9e87e96ee54f52.png)<br>When using in copy, use sentence case. | beta |  
| **analytics** | A feature that allows users to view insights and reports on how their account is performing. | Do not capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. |  |  
| **API** | An application programming interface (API) allows different software applications to communicate with each other. | Capitalize each letter of the abbreviation. OK to use the abbreviation. |  |  
| **app** | Web service or application that connects with Zapier (like Gmail or Slack). | Customer&hyphen;facing term. <br>Use the term “integration or app integration” when on the developer platform. 
 |  |  
| **app connection** | The link between your app account and Zapier. | App connection is clearer for users, but you can use “connection” by itself when space is a factor, or if it is a connection that is not an app. |  |  
| **App Directory** | Zapier's library of available apps and their features. |  |  |  
| **asset** | Building blocks like specific Zaps, Tables, and Interfaces that make up your automation. | Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. |  |  
| **audit log** | Chronological record of events that have occurred within your Zapier account. | Don't capitalize unless it's used at the start of a sentence and in that case, use sentence case “Audit log”. |  |  
| **authentication** | How users securely connect their app accounts to Zapier. | Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. |  |  
| **automation** | Handles routine work for you based on rules you set, without any hands&hyphen;on intervention. | Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. |  |  
| **autoreplay** | A feature that automatically reruns failed Zaps. | Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. |  |  
| **autoreplay override** | Setting that overrides the account&hyphen;wide autoreplay settings for a single Zap.  | Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase. Even when capitalizing, use sentence case. |  |
