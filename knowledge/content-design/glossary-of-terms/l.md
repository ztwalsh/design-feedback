| Term | Definition | Usage Guidelines |  
| **linked assets** | Zapier products (Tables, Canvases, etc) used in a Zap. | **Example:** This Zap includes 10 linked assets.<br>Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase; when capitalized, use sentence case. |  
| **linked record ** | Records from other tables that are used in a table. | **Example:** This table has 3 linked records from another table. <br>Don’t capitalize unless used to start a sentence, heading, title, UI label, or standalone phrase; when capitalized, use sentence case. |
