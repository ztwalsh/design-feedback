# Input fields

---

## Input fields

- Use clear and useful labels, hints, and [placeholder](https://coda.io/d/_dH5LiS8Mxyt/_suolr#_luHyA) (or ghost) text to help users enter accurate information.

- Check that field validation rules are not overly restrictive, especially for personal information (e.g. no minimum character counts, accept accented characters and hyphens in names).

- Choose the right field type

- Use a date picker for dates (not a dropdown menu or text input field).
