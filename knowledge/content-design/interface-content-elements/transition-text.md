# Transition text

---

## Transition/waiting text

- Use transition text to confirm that an action is happening (and that the user isn’t waiting in vain).

- If the action is in progress, use the present continuous tense of the verb (e.g., “is uploading”).

- If the wait is longer than a few seconds, [animation](https://cdn.dribbble.com/users/447957/screenshots/6899626/payment-animation.gif) is helpful.
