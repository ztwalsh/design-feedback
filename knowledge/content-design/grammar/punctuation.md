## Skip periods most of the time

Avoiding unnecessary periods and other punctuation helps the reader scan text more easily. 

Avoid using periods to end single sentences, particularly in:

- Labels

- Tooltip text

- Bulleted lists

- Dialog body text

- Hyperlinked text

Use periods for:

- Multiple sentences

- Long or complex sentences, if it suits the context

- Any sentence followed by a link
